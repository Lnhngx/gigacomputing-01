DDR4 3200MHz Support
The GIGABYTE BRIX provides an upgraded memory controller that supports DDR4 memory at up to 3200MHz. DDR4 is a type of synchronous dynamic random-access memory with a double data rate and a high-bandwidth interface. Its primary advantages over its predecessor, DDR3, include higher module density, lower voltage requirements, and higher data transfer speeds.
